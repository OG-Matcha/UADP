# UADP 快速啟動指南（小白專用）

> **給「財伴 (Cái Bàn)」的非技術開發成員**  
> **最後更新：** 2026-01-04

---

## 🎯 什麼是「零門檻」？

**零門檻** = 你不需要：
- ❌ 會寫程式碼
- ❌ 懂技術術語
- ❌ 自己設定環境
- ❌ 打開終端機輸入指令

**你只需要：**
- ✅ 會雙擊檔案
- ✅ 會點擊「是」或「執行」
- ✅ 會等待幾分鐘

---

## 📸 圖解步驟（Windows）

### 步驟 1：找到啟動檔案

在你的專案資料夾中，找到這個檔案：

```
📁 你的專案資料夾
  └── 📄 UADP-Windows開始.bat  ← 就是這個！
```

**看起來像這樣：**
```
UADP-Windows開始.bat
```

---

### 步驟 2：雙擊啟動

1. **用滑鼠左鍵雙擊** `UADP-Windows開始.bat`

2. 如果 Windows 問你：
   ```
   Windows 已保護你的電腦
   你要執行這個檔案嗎？
   ```
   
   **點擊「更多資訊」**，然後點擊 **「仍要執行」**

---

### 步驟 3：等待自動安裝（如果需要）

**如果這是第一次使用：**

視窗會顯示：
```
[1/4] 檢查 Python 環境...
   Python 未安裝，正在嘗試自動安裝...
   這可能需要幾分鐘，請稍候...
```

**你只需要：**
- ✅ 等待（可能需要 2-5 分鐘）
- ✅ 不要關閉視窗
- ✅ 不要點擊任何東西

**如果安裝成功：**
視窗會顯示：
```
   ✓ Python 安裝完成，請重新啟動此腳本
```

**這時你需要：**
1. 關閉視窗
2. 再次雙擊 `UADP-Windows開始.bat`

---

### 步驟 4：看到「初始化完成！」

當視窗顯示：
```
========================================
✅ 初始化完成！
========================================
```

**恭喜！你已經完成了！**

---

## 📸 圖解步驟（macOS）

### 步驟 1：找到啟動檔案

在你的專案資料夾中，找到這個檔案：

```
📁 你的專案資料夾
  └── 📄 UADP-macOS開始.command  ← 就是這個！
```

---

### 步驟 2：雙擊啟動

1. **用滑鼠左鍵雙擊** `UADP-macOS開始.command`

2. 如果 macOS 問你：
   ```
   無法打開「UADP-macOS開始.command」，
   因為它來自未識別的開發者。
   ```
   
   **解決方法：**
   1. 打開「系統設定」→「隱私權與安全性」
   2. 找到「已阻擋的開發者」區塊
   3. 點擊「仍要開啟」

---

### 步驟 3：安裝開發者工具（如果需要）

**如果這是第一次使用：**

macOS 可能會彈出視窗：
```
需要安裝「命令列開發者工具」嗎？
```

**點擊「安裝」**

然後：
1. 等待下載（可能需要 5-10 分鐘）
2. 安裝完成後，**重新雙擊** `UADP-macOS開始.command`

---

### 步驟 4：看到「初始化完成！」

當視窗顯示：
```
========================================
✅ 初始化完成！
========================================
```

**恭喜！你已經完成了！**

## ❓ 常見問題

### Q1: 視窗閃一下就關閉了？

**A:** 這表示可能有錯誤。解決方法：
1. 打開「命令提示字元」（Windows）或「終端機」（macOS）
2. 進入你的專案資料夾
3. 執行啟動檔案，這樣可以看到錯誤訊息

**Windows：**
```cmd
cd C:\你的專案路徑
UADP-Windows開始.bat
```

**macOS：**
```bash
cd /你的專案路徑
bash UADP-macOS開始.command
```

---

### Q2: 說「找不到 Python」？

**A:** 這表示自動安裝失敗了。解決方法：

**Windows：**
1. 打開瀏覽器
2. 前往：https://www.python.org/downloads/
3. 下載並安裝 Python 3.10 或更新版本
4. 安裝時**記得勾選「Add Python to PATH」**
5. 安裝完成後，重新執行啟動檔案

**macOS：**
1. 打開「終端機」
2. 執行：`xcode-select --install`
3. 等待安裝完成
4. 重新執行啟動檔案

---

### Q3: 說「權限不足」？

**A:** 這表示你需要管理員權限。

**Windows：**
1. 在 `UADP-Windows開始.bat` 上**右鍵**
2. 選擇「以系統管理員身分執行」

**macOS：**
1. 打開「終端機」
2. 執行：`chmod +x UADP-macOS開始.command`
3. 重新雙擊檔案

---

### Q4: 安裝 Python 要等很久？

**A:** 這是正常的！
- 第一次安裝 Python 可能需要 2-5 分鐘
- 請耐心等待，不要關閉視窗
- 安裝完成後，之後就不需要再安裝了

---

## ✅ 完成後，下一步是什麼？

當你看到「初始化完成！」後：

1. **關閉啟動視窗**

2. **打開 Cursor IDE**
   - 如果還沒安裝，請前往：https://cursor.sh/

3. **在 Cursor 中開啟你的專案資料夾**

4. **告訴 AI：**
   ```
   [MODE: ARCHITECT] 我想做一個 [你的專案想法]
   ```
   
   例如：
   ```
   [MODE: ARCHITECT] 我想做一個記帳 App
   ```

5. **AI 會開始問你問題，用你自己的話回答就好！**
