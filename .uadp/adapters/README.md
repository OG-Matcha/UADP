# UADP 適配器目錄

> 此目錄存放技術棧適配器模板，用於引導 AI 學習特定技術棧的最佳實踐。

---

## 適配器結構

每個適配器應包含：
- **適配器規則檔案**（`.mdc` 或 `.md`）：定義技術棧特定的開發規範
- **測試框架配置**：指定測試框架與命令
- **最佳實踐指南**：技術棧特定的編碼標準

---

## 預設適配器

### mobile-flutter
Flutter 行動應用開發適配器

### web-modern
現代 Web 開發適配器（React/Vite）

### backend-api
後端 API 開發適配器（Node.js/Python）

---

## 使用方式

適配器由 UADP 框架自動偵測或透過 [DIAGNOSIS] 階段對話引導選擇。

AI 會讀取適配器模板，學習該技術棧的：
- 最佳實踐
- 測試命令
- 編碼規範
- 專案結構
